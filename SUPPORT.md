# Support

## Getting Help

1. Open an issue: `https://github.com/petalo/twenty-gmail-extension/issues`
2. Include reproducible steps and non-sensitive logs.
3. Use the bug template for runtime failures in Gmail.

## Before Opening an Issue

1. Read [INSTALL.md](INSTALL.md) and [UPDATE.md](UPDATE.md).
2. Check troubleshooting guidance in [docs/runbooks/04-troubleshooting.md](docs/runbooks/04-troubleshooting.md).
3. Confirm no API tokens or private message content are included in the report.
